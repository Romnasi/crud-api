# crud-api
"CRUD" service build on pure Node js

---

## Build and Run

install dependencies

```bash
npm i
```

start app

```bash
npm run start
```

---

### Endpoints

#### User Service

|HTTP Method|URL|Description|
|---|---|---|
|`GET`|<http://localhost:4000/api/users/> | Get all Users |
|`GET`|<http://localhost:4000/api/users/{userId}> | Get User by ID |
